## Thévenin Resistance
### Steps

### 1. Find $V_{Th}$ (Open-Circuit Voltage)
*   **Remove the load resistance $R_L$** between terminals $a$ and $b$.
*   **Identify the controlling variable:**
    
    *   Locate the variable that controls the dependent source (e.g., $v_1$).
        
    *   Express it in terms of the node voltage at terminal $a$ ($V_a$) and the source voltage.
        
    *   _Example:_ $v_1 = 12 - V_a$.
        
*   **Apply Nodal Analysis (KCL):**
    
    *   Sum the currents leaving node $a$ and set them to zero.
        
    *   Example: $\frac{V_a - 12}{4} - 4 + \frac{V_a + 2v_1}{2 + 4} = 0$.

### 2. Find $I_{sc}$ (Short-Circuit Current)

*   **Place a short circuit** directly between terminals $a$ and $b$.
    
*   **Force $V_a = 0$:** Because node $a$ is now connected to ground (node $b$), $V_a$ becomes $0 \text{ V}$ automatically.
    
*   **Recalculate the controlling variable:**
    
    *   Since $V_a = 0$, the variable changes (e.g., $v_1 = 12 - 0 = 12 \text{ V}$).
        
*   **Calculate $I_{sc}$ using KCL:**
    
    *   Sum the currents leaving node $a$, including the current $I_{sc}$ flowing through the short.
        
    *   _Example:_ $I_{sc} + \frac{0 - 12}{4} - 4 + \frac{0 + 2(12)}{6} = 0 \Rightarrow I_{sc} = 3 \text{ A}$.
        

### 3. Calculate $R_{Th}$ and Max Power

*   **Find Thévenin Resistance:** Use the ratio $R_{Th} = \frac{V_{Th}}{I_{sc}}$.
    
    *   _Example:_ $R_{Th} = \frac{36}{3} = 12 \text{ }\Omega$.
        
*   **Maximum Power Condition:** Set $R_L = R_{Th}$.
    
*   **Calculate $P_{max}$:** Use the formula $P_{max} = \frac{V_{Th}^2}{4R_{Th}}$.
    
    *   _Example:_ $P_{max} = \frac{36^2}{4(12)} = 27 \text{ W}$.
