# ELEN2000 EXAM CRACKER

### 1. CIRCUIT FUNDAMENTALS & TRICKS
* [cite_start]**Node Voltage (Nodal):** Sum of currents leaving node = 0. [cite_start]If a voltage source is between two non-reference nodes, use a **Supernode**.
* [cite_start]**Mesh Current:** Sum of voltages around loop = 0. [cite_start]If a current source is on a shared branch, use a **Supermesh**.
* **Power Check:** $P_{absorbed} = VI$ (current enters +). $P_{delivered} = -VI$. [cite_start]$\sum P = 0$.
* **Delta-Wye ($\Delta-Y$):** $R_Y = \frac{\text{Product of adjacent } \Delta}{\sum \Delta}$. [cite_start]$R_\Delta = \frac{\sum \text{Products pair-wise } Y}{\text{Opposite } Y}$[cite: 1, 5].

### 2. THEVENIN & NORTON (THE PROCESS)
* **Step 1:** Remove load $R_L$. [cite_start]$V_{th} = V_{oc}$ (open circuit voltage)[cite: 1, 5].
* [cite_start]**Step 2 ($R_{th}$):** Turn off independent sources (V-source=short, I-source=open)[cite: 1, 5].
* **Step 3 (Dependent Sources):** If present, you **must** use $R_{th} = V_{oc} / I_{sc}$ or apply a 1V test source and find $I$[cite: 1, 5].
* **Max Power Transfer:** Occurs when $R_L = R_{th}$. [cite_start]$P_{max} = \frac{V_{th}^2}{4R_{th}}$[cite: 1, 5].

### 3. CAPACITORS (C) & INDUCTORS (L)
* **Energy:** $W_L = \frac{1}{2}Li^2$ | $W_C = \frac{1}{2}Cv^2$[cite: 2, 5].
* **Continuity:** $v_C(0^-) = v_C(0^+)$ and $i_L(0^-) = i_L(0^+)$. [cite_start]They cannot change instantaneously[cite: 2, 5].
* **DC Steady State:** $L \to$ Short Circuit ($v=0$). [cite_start]$C \to$ Open Circuit ($i=0$)[cite: 2, 5].

### 4. FIRST ORDER (RL/RC) - STEP BY STEP
* **General Eq:** $x(t) = x(\infty) + [x(0) - x(\infty)]e^{-t/\tau}$[cite: 3, 5].
* **RC:** $\tau = R_{eq}C$ | [cite_start]**RL:** $\tau = L/R_{eq}$[cite: 3, 5].
* [cite_start]**Process:** 1. Find $x(0^-)$ to get $x(0^+)$[cite: 3]. 2. [cite_start]Find $x(\infty)$ in final state[cite: 3]. 3. [cite_start]Find $R_{eq}$ looking from $L/C$ terminal at $t>0$[cite: 3].
* [cite_start]**Unit Step $u(t)$:** $0$ for $t<0$, $1$ for $t>0$[cite: 3, 5].

### 5. SECOND ORDER (RLC) - THE KILLER SECTION
* **$\alpha$ (Damping):** Series $= \frac{R}{2L}$. Parallel $= \frac{1}{2RC}$[cite: 4, 5].
* [cite_start]**$\omega_0$ (Resonant):** $\frac{1}{\sqrt{LC}}$ for both[cite: 4, 5].
* **Damping Ratio $\zeta = \alpha / \omega_0$:** [cite: 4, 5]
    * $\zeta > 1$: **Overdamped**. $s_{1,2} = -\alpha \pm \sqrt{\alpha^2 - \omega_0^2}$. $x(t) = A_1e^{s_1t} + A_2e^{s_2t}$[cite: 4, 5].
    * $\zeta = 1$: **Critically**. $s_{1,2} = -\alpha$. [cite_start]$x(t) = (A_1 + A_2t)e^{-\alpha t}$[cite: 4, 5].
    * $\zeta < 1$: **Underdamped**. $\omega_d = \sqrt{\omega_0^2 - \alpha^2}$. $x(t) = e^{-\alpha t}(B_1\cos \omega_d t + B_2\sin \omega_d t)$[cite: 4, 5].
* **Initial Conditions:** You usually need $x(0)$ and $x'(0)$. [cite_start]Note: $v_L = L\frac{di}{dt}$ and $i_C = C\frac{dv}{dt}$.

### 6. PRACTICE TEST MISTAKE PREVENTERS
* **Polarity:** Always draw current directions. If Nodal gives a negative $V$, it just means $V < V_{ref}$.
* **Time Switching:** At $t=0$, if a switch opens/closes, the circuit topology changes. Calculate a new $R_{eq}$ for $\tau$[cite: 3].
* [cite_start]**Complex Roots:** In underdamped cases, ensure your calculator is in **Radians** for $\omega_d t$ calculations.
